smolagents Starter

A compact Python starter for building web-search agents that think in code, call real tools, and run through an interactive CLI.

View Repository Start Building

Small enough to understand. Real enough to extend.

This starter keeps the agent loop visible: load a Nebius API key, connect an OpenAI-compatible model, attach web search, and start asking questions. It is designed for fast experiments, clean demos, and practical agent prototypes.

Quickstart

Run the agent from your terminal.

Install

git clone https://github.com/tirth1263/smolagents-starter.git
cd smolagents-starter
pip install -r requirements.txt

Configure

cp .env.example .env
# then set NEBIUS_API_KEY

Use

python main.py

Architecture

The starter has one job: make the first useful agent obvious.

Code-first reasoning

`CodeAgent` can solve tasks by writing Python and using the tools attached to it.

Fresh web context

`DuckDuckGoSearchTool` gives the agent a simple path to current information for research prompts.

Hosted inference

`OpenAIServerModel` points at Nebius Token Factory through its OpenAI-compatible API.

Example prompts

Ask questions that benefit from search.

  • Who won the latest F1 race and by how many seconds?
  • Summarize today's top story on Hacker News.
  • Find three recent papers about agentic RAG.
